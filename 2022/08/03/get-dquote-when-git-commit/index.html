Get "dquote>" when executing "git commit"

Problem

When I executed git commit on the Mac terminal, I got dquote> showed below:

What’s wrong

That was because I used Chinese punctuation mark ” at the end of the commit but not the English style ".

Solution

There are two simple ways to fix that problem:

  1. Ctrl+C to quit editing, but need to re-enter the same command

  2. Just enter a " and go ahead
