Use MOV or GIF file for demo in your GitHub README.md

I use MOV and GIF files to demonstrate demos for my projects. Here I will introduce how to generate and embed them into GitHub README.md.

GIF demo

  1. Record a video that you use to convert into GIF.

  2. Search “Gifski“ in App Store and click GET.

    There are also other useful online tools I use for GIFs processing:

    • GIFGIFS
    • Free Online Image Editor

  3. Open or drag the video into Gifski.

  4. Configure the GIF being generated as you need.

  5. Place the GIF in a demo folder of your project.

  6. In README.md mention the GIF.

MOV demo

  1. Record a MOV file that you use for demo.

  2. Click on README.md file in GitHub.

  3. Click “Edit this file“ button.

  4. Drag the MOV file into README.md edit section.

  5. Wait for generating the URL of MOV file.

  6. Preview and check the demo.

  7. Commit changes.
